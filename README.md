# let、const、var 的区别有哪些？

let 和 var 声明一个变量, const 声明一个只读变量 只保证指针不能动  一旦声明必须立即初始化 否则报错  

let 和 const 声明不存在变量提升 只在声明的块作用域有效    var 存在变量提升

let 和 const 存在暂时性死区 声明之前使用会报错 

let 和 const 在同一个块作用域（ES6规定 块级作用域必须有大括号）  重复声明会报错
